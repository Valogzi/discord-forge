## Global CLI Options

| Option | Description | Default |
| --- | --- | --- |
| <Code>--name &lt;name&gt;</Code> | Set the project name | `discord-forge` |
| <Code>--template &lt;template&gt;</Code> | Set the template to use | `default` |
| <Code>--typescript</Code>, <Code>-t</Code> | Use TypeScript instead of JavaScript | `false` |
| <Code>--dependencies</Code>, <Code>-d</Code> | Install dependencies after scaffolding | `true` |
| <Code>--engine &lt;engine&gt;</Code> | Set the engine to use | `default` |
| <Code>--version</Code> | Show CLI version | `0.1.0` |
