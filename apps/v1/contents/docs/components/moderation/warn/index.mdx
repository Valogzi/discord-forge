## Installation Steps

This is the steps to add the warn component to TypeScript or JavaScript template

---

With npm:

```bash
npx create-discord-forge@latest add warn
```

with pnpm:

```bash
pnpx create-discord-forge@latest add warn
# or
pnpm dlx create-discord-forge@latest add warn
```
