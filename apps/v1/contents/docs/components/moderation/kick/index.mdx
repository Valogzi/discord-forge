## Installation Steps

This is the steps to add the kick component to TypeScript or JavaScript template

---

With npm:

```bash
npx create-discord-forge@latest add kick
```

with pnpm:

```bash
pnpx create-discord-forge@latest add kick
# or
pnpm dlx create-discord-forge@latest add kick
```
