## Installation Steps

This is the steps to add the ban component to TypeScript or JavaScript template

---

With npm:

```bash
npx create-discord-forge@latest add ban
```

with pnpm:

```bash
pnpx create-discord-forge@latest add ban
# or
pnpm dlx create-discord-forge@latest add ban
```
